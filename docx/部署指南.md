## 项目结构

```
sys1/
├── server/          # Spring Boot 后端（Java 17）
├── client/          # Vue3 + Element Plus 管理后台
├── uniapp/          # uni-app 微信小程序

```

|------|-------------|

## 一、Server（后端）

### 1.1 配置

| 文件 | 用途 |
|------|------|
| `application.properties` | 公共配置 + `spring.profiles.active=dev` |
| `application-dev.properties` | 开发数据库 + DEBUG 日志 |
| `application-prod.properties` | 生产数据库 + WARN 日志 |

### 1.2 构建

```bash
cd server
mvn clean package -DskipTests
```

生成的 jar 包：`target/server-0.0.1-SNAPSHOT.jar`

### 1.3 运行

**开发环境（默认）：**
```bash
java -jar ./target/server-0.0.1-SNAPSHOT.jar
```

**生产环境：**
```bash
java -jar ./target/server-0.0.1-SNAPSHOT.jar --spring.profiles.active=prod
```

### 1.4 生产数据库配置

`application-prod.properties` 中的数据库连接：

## 二、Client（管理后台）

### 2.1 配置

| 文件 | 环境 | 命令 |
|------|------|------|
| `.env.development` | 本地开发 | `npm run dev` |
| `.env.production` | 生产部署 | `npm run build` |
| `.env.production-local` | 本地连生产 | `npm run dev:prod` |

### 2.2 本地运行

```bash
cd client
npm install

# 连本地后端
npm run dev              # 访问 http://localhost:5173

# 连生产后端（走代理，无跨域）
npm run dev:prod         # 访问 http://localhost:5173
```

### 2.3 构建生产包

```bash
npm run build
```

产物在 `dist/` 目录。

### 2.4 部署到 DevBox / Nginx

```bash
# 上传
scp -r dist/ root@你的服务器:/home/devbox/project/client/

# 在服务器上配置 Nginx
```

**Nginx 配置参考：**
```nginx
server {
    listen       80;
    server_name  wkctqvunclym.sealosbja.site;

    root   /home/devbox/project/client/dist;
    index  index.html;

    # 前端静态页面
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 转发到后端
    location /api/ {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 图片资源转发
    location /uploads/ {
        proxy_pass http://localhost:8080;
    }
}
```

## 三、Uniapp（微信小程序）

### 3.1 配置

编辑 `utils/config.js`，切换环境：

```js
// 开发环境
const ENV = 'dev'    // → http://localhost:8080

// 生产环境
const ENV = 'prod'   // → https://wkctqvunclym.sealosbja.site
```

### 3.2 开发运行

1. 用 HBuilderX 打开 `uniapp/uniapp/` 目录
2. 修改 `config.js` 中 `ENV` 为 `dev`
3. 运行 → 微信开发者工具

### 3.3 发布上线

1. 修改 `config.js` 中 `ENV` 为 `prod`
2. HBuilderX → 发行 → 小程序-微信
3. 微信开发者工具 → 上传代码
4. 微信公众平台 → 提交审核
